# init-express-4

Initial files for creating a node.js with Express 4 app.

## Install

`npm i && bower i` to pull dependencies

# Usage

run `gulp` for development mode or `gulp --live` to build optimised version

## Features
- browser-sync
- uncss
- css autoprefixing
- live reload
- etc.
